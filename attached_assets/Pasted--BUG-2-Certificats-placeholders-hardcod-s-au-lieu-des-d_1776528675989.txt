### BUG-4 : Ordonnance — "Qté: null comprimé" affiché

Dans `/ordonnances/1/imprimer` :
"Ypozane 15 mg — 15 mg — — — pendant — Qté: null comprimé"

**Fix requis** :
1. Dans le composant qui formate les lignes d'ordonnance, remplacer :
```tsx
// AVANT
`Qté: ${ligne.quantite} ${ligne.unite}`
// APRÈS
ligne.quantite ? `Qté: ${ligne.quantite} ${ligne.unite ?? 'comprimé'}` : ''
```
2. Aussi masquer les champs vides ("— — —" entre deux tirets consécutifs = champ null) :
```tsx
const formatLigneOrdonnance = (ligne) => {
  const parts = [
    ligne.nomMedicament,
    ligne.dosage ? `${ligne.dosage} ${ligne.uniteDosage ?? ''}` : null,
    ligne.voieAdmin,
    ligne.frequence,
    ligne.duree ? `pendant ${ligne.duree}` : null,
    ligne.quantite ? `Qté: ${ligne.quantite} ${ligne.unite ?? 'comprimé(s)'}` : null,
  ].filter(Boolean)
  return parts.join(' — ')
}
```

### BUG-7 : SQL — Nettoyage BDD et kétamine stupéfiant

Exécuter ces requêtes SQL sur la base de données :

```sql
-- Nettoyer les factures fantômes 0€ (si pas encore fait)
DELETE FROM lignes_facture WHERE facture_id IN (
  SELECT id FROM factures WHERE montant_ttc = 0 AND statut = 'en_attente'
);
DELETE FROM factures WHERE montant_ttc = 0 AND statut = 'en_attente';

-- Ajouter kétamine et butorphanol comme stupéfiants
UPDATE stock SET est_stupefiant = true 
WHERE LOWER(nom_commercial) LIKE '%ketamine%' 
   OR LOWER(nom_commercial) LIKE '%kétamine%'
   OR LOWER(nom_commercial) LIKE '%butorphanol%'
   OR LOWER(nom_commercial) LIKE '%morphine%';

-- Insérer kétamine si absente
INSERT INTO stock (nom_commercial, categorie, est_stupefiant, stock_actuel, stock_min, stock_max, unite)
SELECT 'Kétamine 100mg/ml inj', 'Médicament', true, 5, 2, 20, 'flacon'
WHERE NOT EXISTS (SELECT 1 FROM stock WHERE LOWER(nom_commercial) LIKE '%ketamine%');
```
