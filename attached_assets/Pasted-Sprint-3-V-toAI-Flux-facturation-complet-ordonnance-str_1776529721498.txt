### BLOC A : Flux facturation bout-en-bout (critique business)

Objectif : depuis une consultation terminée → créer actes → créer facture → encaisser.

**A1 — Bouton "Finaliser et facturer" en fin de consultation**
Sur l'onglet "Ordonnance & Actes" quand la consultation a au moins 1 acte :
1. Ajouter un bouton bleu "Finaliser et facturer" en bas de page
2. Ce bouton appelle POST /api/factures avec :
   - `consultationId`
   - `patientId` (depuis la consultation)
   - `lignes` (actes de la consultation)
   - Calcule automatiquement `totalHt`, `tva`, `montantTtc`
3. Redirige vers `/factures/[newId]`

**A2 — Route POST /api/factures depuis actes consultation**
Si pas encore implémenté :
```typescript
router.post('/api/factures', async (req, res) => {
  const { consultationId, patientId } = req.body
  const actes = await db.select().from(actesTable).where(eq(actesTable.consultationId, consultationId))
  if (!actes.length) return res.status(400).json({ error: 'Aucun acte — impossible de facturer' })
  
  const lignes = actes.map(a => ({
    description: a.description,
    quantite: a.quantite,
    prixUnitaireHt: a.prixUnitaireHt,
    tauxTva: a.tauxTva ?? 20,
    total: a.quantite * a.prixUnitaireHt * (1 + (a.tauxTva ?? 20) / 100)
  }))
  const totalHt = lignes.reduce((sum, l) => sum + l.quantite * l.prixUnitaireHt, 0)
  const montantTtc = lignes.reduce((sum, l) => sum + l.total, 0)
  
  const numero = await generateFactureNumero() // FACT-2026-XXXX
  const [facture] = await db.insert(facturesTable).values({
    consultationId, patientId, lignes, totalHt, montantTtc, statut: 'en_attente', numero
  }).returning()
  
  res.json(facture)
})
```

**C2 — Route POST /api/rappels depuis consultation**
```typescript
router.post('/api/rappels', async (req, res) => {
  const { consultationId, patientId, label, joursDelai } = req.body
  const dateEcheance = new Date()
  dateEcheance.setDate(dateEcheance.getDate() + joursDelai)
  const rappel = await db.insert(rappelsTable).values({
    consultationId, patientId, label, dateEcheance, statut: 'actif'
  }).returning()
  res.json(rappel[0])
})
```
