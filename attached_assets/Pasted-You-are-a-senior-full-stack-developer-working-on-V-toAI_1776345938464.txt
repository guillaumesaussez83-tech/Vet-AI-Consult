━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Create artifacts/api-server/src/routes/agenda/index.ts

CORE FUNCTION getAvailableSlots(vetId, date, clinicId, slotDuration=20):
1. Get semaine_type for this day of week → if none, return []
2. Check exceptions_planning covering this date:
   - conge/maladie/formation/fermeture → return []
   - garde_exceptionnelle → use override hours
3. Check rotations_weekend: if Sat/Sun and vet not in rotation → return []
4. Generate slots between heure_debut and heure_fin, skip pause window
5. Remove slots overlapping existing rendez_vous
6. Return [{ heure: "09:00", disponible: true }, ...]

ROUTES:

GET /api/agenda/veterinaires
→ All active vets for clinic with couleur and initiales

GET /api/agenda/slots/:vetId/:date
→ Available slots for one vet on one date

GET /api/agenda/slots/multi/:date
→ Available slots for ALL vets on a date
→ { [vetId]: { vet, slots, travaille: boolean } }

GET /api/agenda/rendez-vous/semaine/:dateDebut
→ All rdv for the week, grouped by day then by vet, include vet couleur

GET /api/agenda/planning/mois/:annee/:mois
→ For each day × each vet: { travaille, nb_rdv, exception, est_garde_weekend }

POST /api/agenda/rendez-vous
→ Validate slot available → if taken return 409 with next available slots
→ Create rdv with statut_salle='en_attente_arrivee'

PUT /api/agenda/rendez-vous/:id
→ Validate new slot if time/vet changed → update

PUT /api/agenda/rendez-vous/:id/statut-salle
→ Update statut_salle only (for kanban drag & drop)

POST /api/agenda/planning/exception
→ Create exception, check for rdv conflicts
→ If conflicts: return { conflits: [{rdv_id, date, heure, animal, proprietaire}] }

DELETE /api/agenda/planning/exception/:id

POST /api/agenda/planning/semaine-type
→ Upsert planning_semaine_type for vet+day

POST /api/agenda/rotations/generer
→ Input: { vetIds, dateDebut, dateFin, cycle, confirmer }
→ Generate fair rotation (vet1 week1, vet2 week2...), preview if confirmer=false, save if true

GET /api/agenda/prochain-creneau/:vetId?
→ Find next available slot for one vet or any vet
